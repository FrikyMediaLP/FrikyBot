Twitch Login

Using Twitch to Login authenticates your Twitch ID to the Server. You Userlevel (your access level) is set by an admin or defaulted as a viewer.

To Authenticate you have to login and give us access to request an ID TOKEN or an ACCESS TOKEN

  • ID TOKEN: Authenticates that YOU just logged in. Only valid for 15 Minutes.
  • ACCESS TOKEN: Authorizes the Website Client (not the server) to access (some) actions on behalf of your account (e.g. send Chat Messages). This Token remains valid for a while (up to 30 days) or until you remove the Bot Connection in your Twitch Settings.

For Authentication to work across sites, you NEED to allow Cookies to be used! Otherwise, the Website just doesnt know who you are! Read Cookie Usage
